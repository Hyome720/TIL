# 프로그래밍이란?

## 1. 정의

- 'program' + '(m)ing'
- 프로그램을 만드는 행위
  - 특정 작업을 수행하는 일련의 명령어들의 모음

- 컴퓨터 -> 일 에게 명령

- 프로그래머 == 프로그램을 만드는 사람

  

## 2. 언어

- 생각을 표현 및 전달하기 위한 체계
- 문법적으로 맞을 때 언어 공동체 내에서 '이해될 수 있는' 말의 집합
- 컴퓨터 - 기계어
  - 2진법 - 0과 1로 모든 것을 표현
  - 0 = off, 1 = on
- 기계어의 대안 => 프로그래밍 언어
